# Multivariate Time Series Forecasting using Deep Learning

## Overview

This project focuses on multivariate time series forecasting using deep learning models.  
It is built upon the **iTransformer codebase** as a framework and extended with additional models, including recurrent architectures and a custom proposed model.

## Contributions

- Integration of deep learning models (LSTM, RNN, GRU)
- Development of a custom forecasting model
- Comparative experiments on multiple architectures

## Notes

This work is part of a master thesis and follows a standardized experimental pipeline for fair evaluation.  
The experiments and model implementations leverage the modular structure provided by the iTransformer framework.
